How to install

  1. Use Chrome, Edge, or another Chromium-based browser.
  2. Connect the device you want to flash to your computer via USB.
  3. Click the appropriate Flash button above.
  4. Select the serial port when prompted.
  5. Wait for the flash to complete, then reconnect power.
